Lancez les fichiers .sh avec la commande bash de WSL.

###########################################################################################

fill_db.sh créée la base de donnée et la remplit.
destroy_db.sh supprime toutes les tables et données de la base de donnée.
launch_app.sh permet de lancer l'application web, une fois la commande exécutée, il suffit d'ouvrir une page web sur l'adresse : 127.0.0.1:5000
